项目说明

使用要求

1. 引用第三方类库要和组内同事或者项目管理者建议是否使用，避免过多使用造成项目资源过大，后面同事使用不清楚等问题。

2. 引用需要对应添加使用方法和被使用的功能模块。

角命令大全

ng new project     新建项目

npm install     安装项目需要的包

ng serve    启动项目

ng g c User     参加组件（user组件名）

ng serve --prod --aot     最小化打包启动

ng build --prod --aot     编译打包
